How to use & file format
  • CSV: two columns time, intensity — no header needed
  • NPY: 1D array (intensity only) or 2D array shape (N,2)
  • Demo: generates a synthetic dispersion-stretched Gaussian with simulated cell modulation

Plots produced:

  • Time-domain waveform + Hilbert envelope
  • Power spectral density |FFT|²
  • STFT spectrogram with peak-frequency tracking (chirp rate)
  • TD-GS phase preview (one Gerchberg-Saxton iteration, D1=−600 ps², D2=−1200 ps²)
  • Instantaneous frequency (Hilbert)
  • Autocorrelation R(τ)

Run locally: python optical_dashboard/app.py → localhost:5000

Claude Code local setup (actual steps)

1. Install Node.js 18+ (from nodejs.org)

2. Install the CLI:

npm install -g @anthropic-ai/claude-code

3. Set your API key:

set ANTHROPIC_API_KEY=sk-ant-api03-...   (Windows cmd)

export ANTHROPIC_API_KEY=sk-ant-api03-...   (bash/zsh)

4. Launch:

claude

Or in your project directory: cd D:\Summer2026\Dispersion-Assisted-GS-Phase-Recovery && claude

Get API key: console.anthropic.com

Docs: docs.anthropic.com/claude-code
